# NôsCidade – Urban AI Platform

Building the first AI-powered urban intelligence infrastructure 
for emerging coastal cities, starting with Mindelo, Cabo Verde.

## Vision
Transform Mindelo into a replicable smart coastal city model using:
- Artificial Intelligence
- Urban Data Science
- Participatory Digital Governance
- Climate Resilience Modeling
